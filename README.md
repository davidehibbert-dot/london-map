# London Interactive Map

This repository hosts an interactive Leaflet.js map of London with major attractions, parks, museums, cathedrals, markets, and Underground stations.

## How to View

1. Go to the GitHub Pages link for this repository, which will look like:

   ```
   https://yourusername.github.io/reponame/london-map.html
   ```

   Replace `yourusername` and `reponame` with your GitHub details.

2. The map will open in your browser and is fully interactive.

## How to Update

- To change the map (e.g. add/remove places), edit `london-map.html` and commit the changes.
- GitHub Pages will automatically update within a minute or two.
